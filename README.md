# くもにtpするだけのプラグイン (制作:20分)

## 概要

くもにTPするだけのプラグインです。  
くもにTPしたあとは5秒間透明化されます。

## 使い方

- /kumo  
くもへTP (LuckPermsで制限しない限りだれでも)
- /kumo_off
TP OFF! (くもだけ)
- /kumo_on
TP ON!! (くもだけ)

## 詳しいこと

### パーミッション

- `kumo.use`: /kumoを使えるようにする
- `kumo.admin`: /kumo_off, /kumo_onを使えるようにする

### コンフィグ

```
kumo:
  playerName: kumo_0621 # プレイヤーの名前
  invisibilityDuration: 5 # 透明化の時間
```
